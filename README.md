# Telecom Customer Retention Intelligence Platform

## Project Overview

This project aims to develop an end-to-end data science solution that predicts telecom customer churn and helps businesses identify high-risk customers before they leave.

## Objectives

- Analyze telecom customer data
- Perform data preprocessing
- Build and compare machine learning models
- Explain model predictions
- Develop an interactive Streamlit application

## Technologies

- Python
- Pandas
- NumPy
- Matplotlib
- Scikit-learn
- SHAP
- Streamlit
- Git & GitHub
